Haxe 3.0 and ActionScript 3.0

A simple project highlighting the differences between Haxe 3.0 and ActionScript 3.0
July 28, 2013

Project: https://github.com/adireddy/as3haxe/

This project is just to give an idea about all the major differences between Haxe and ActionScript to help new comers to dive into the awesome world of Haxe. I truly believe that Haxe is the future for all the flash developers which helps you to build multi-platform applications.

This project was quiclky put together by refering to the cheat sheet and at the same time correcting few things making it compatible with Haxe 3.0 and OpenFL.

You can setup Haxe 3.0 and OpenFL by refereng to this tutorial.

Also if you love FlashDevelop you can set it up using this tutorial. (Note that you need to download developer version of FlashDevelop)
